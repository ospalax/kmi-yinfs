# KMI/YINFS project

The personal portfolio webapp for the class KMI/YINFS.

Technical stack:

- docker
- POSIX shell
- python3
- django
- PostgreSQL
- HTML5/CSS3
- Javascript

Usage:

1. Edit `docker-compose.yml` (to change usernames, passwords etc.)
2. Build and run it: `# docker-compose up -d`
3. Go to http://localhost:8000

Enjoy!
